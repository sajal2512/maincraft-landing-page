# 🚀 MainCrafts Technology — Landing Page

A modern, premium landing page built for **MainCrafts Technology** using **pure HTML, CSS, and JavaScript** — no frameworks, no dependencies.

## ✨ Features

- **Responsive Design** — Fully responsive layout that works seamlessly on mobile, tablet, and desktop
- **Glassmorphism UI** — Modern frosted-glass card design with backdrop blur effects
- **Smooth Scroll Animations** — Intersection Observer-powered reveal animations on scroll
- **Sticky Navigation** — Header transitions with scroll-aware background and shadow
- **Mobile Hamburger Menu** — Accessible, animated toggle menu for smaller screens
- **Scroll-to-Top Button** — Smooth-scroll back-to-top with visibility toggle
- **Accessibility First** — Semantic HTML, ARIA labels, skip-link, keyboard-navigable, and `prefers-reduced-motion` support
- **SEO Optimized** — Proper meta tags, heading hierarchy, and semantic elements

## 🛠️ Tech Stack

- **HTML5** — Semantic markup with accessibility best practices
- **CSS3** — Custom properties, Flexbox, CSS Grid, animations, glassmorphism
- **Vanilla JavaScript** — Intersection Observer, scroll listeners, mobile menu toggle
- **Google Fonts** — Inter (300–900 weights)
- **Font Awesome 6** — Icon library via CDN

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/sajal2512/maincraft-landing-page.git
   cd maincraft-landing-page
   ```

2. **Open in browser**
   ```bash
   # Simply open the HTML file — no build step needed!
   open index.html
   ```

   Or use a live server:
   ```bash
   npx serve .
   ```
